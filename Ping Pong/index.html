Ping Pong

Lets play some ping pong, without paddles!!

This version takes the user's input and evaluates the given number without looping through the rest of the numbers preceeding it

Rules:

  1. Numbers divisible by 3 become "ping"
  2. Numbers divisible by 5 become "pong"
  3. Numbers divisible by 3 and 5 become "pingpong"

Ping Pong

This version takes the user's input and evaluates it and loops from number 1 to the given input value

Rules:

  1. Numbers divisible by 3 become "ping"
  2. Numbers divisible by 5 become "pong"
  3. Numbers divisible by 3 and 5 become "pingpong"
